Writer's Notebook

A Gist clone which allows writers to post, share, and comment on ideas.

This is the largest project I have done, and I encountered many challenges. I also had to learn many new techniques. Check it out on GitHub

  • Polymorphic notifications (not present in Gist)
  • Counter caches and other optimizations to minimize database queries
  • User messaging (not present in Gist)
  • Email validation
  • Social Media features, including Stars(favorites), comments and forking(copying) posts
  • RESTful Rails controllers
  • Public and Private posts

Asteroids.js

A clone of the popular arcade game, Asteroids

This project was my largest javascript-based program. It was also my best work with a visual representation. Check it out on GitHub

  • Made in JavaScript with HTML5 Canvas
  • Vector Algebra calculates movement and momentum
  • Keymaster converts user input to degree/radian direction
  • Objects loop on and off the screen
  • Collision Detection

Towers of Hanoi

A CSS/HTML5/Jquery representation of the traditional Children's game

I wanted to improve my knowledge of CSS, so I created this game as an exercise. Check it out on GitHub

  • Made in HTML with JQuery
  • Responds to User Input using on click events
  • Gives feedback via an alert if a move is invalid
  • Displays "disks" by swapping the CSS class of HTML5 elements
  • Utilzies highlight effects to change the background color on mouse-over

  • Learn Ruby

    by Chris Pine

    This book is an amazing intro-level text. It doesn't cover a lot of advanced material, but great for beginners.

  • Beginning Ruby

    by Peter Cooper

    Intermediate-level text which really built on everything I learned from Chris Pine's book, and more.

  • Learning SQL

    by Alan Beaulieu

    SQL is a basic foundation, and this is a simple and straightforward way to learn the concept. It didn't offer much practice though.

  • SQLZ00

    Amazing site for SQL practice

    The problems on SQLZOO are deliberately difficult and make you really need to think to solve the problems. Amazing for anyone wanting to brush up and practice their SQL.

  • Codecademy

    An intro to many languages

    I used this site to practice Ruby, HTML, SQL, Javascript, JQuery, and CSS, and I'll continue to use it in the future!

  • Ruby Monk

    A site with a bit different Ruby technique

    When I was having trouble grasping the technicalities of complex Ruby, I used this site for a different take on it. It gives interactive feedback, and keeps lessons lighthearted.

  • Rails Tutorial

    by Michael Hartl

    A bit confusing at first, but a great intro to building a webpage with rails. Also a good look at the MVC pattern. VERY easy to use in the web-based version, as links go directly to the section they reference.

  • Effective JavaScript

    by David Herman

    This book takes the confusing part out of JavaScript. It points out simple techniques and tricks you can use to make your work with JavaScript understandable, productive, and fun!

  • Secrets of the Javascript Ninja

    by John Resig and Bear Bibeault

    Builds off of existing JavaScript knowledge and aims to increase proficiency. It helps the reader to understand libraries.

  • Backbone.js on Rails

    An intro to the Backbone.js library

    An introduction to Backbone.js and how embedded JavaScript can work wonders for your site. A very comprehensive breakdown of the differences between Rails and Backbone.js.

  • App Academy

    App Academy is an immersive, full-time, web development program in New York City. It is an exclusive program which only accepts the top 5% of applicants. I built on what I had learned, and more. They taught me how to code WELL. I received real feedback from people who had worked as web developers. My code is more effecive and more efficient since.
